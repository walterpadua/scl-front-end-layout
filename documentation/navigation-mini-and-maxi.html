Mini and Maxi

For small device screen sizes, the navigation will only be visible if activated by tapping the hamburger button.
This is especially handy for distraction-free mobile reading.

To start pages with maximal width navigation by default, just add class .mn--max to the #root. 

<body>
    <div id="root" class="root mn--max">
        ...
    </div>
</body>

or .mn--min to start the page with minimal navigation by default.

<body>
    <div id="root" class="root mn--min">
        ...
    </div>
</body>
